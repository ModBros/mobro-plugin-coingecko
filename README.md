**CoinGecko plugin for MoBro**

Integrate real-time cryptocurrency metrics from [CoinGecko](https://www.coingecko.com/) into
[MoBro](https://mobro.app).  
This plugin leverages the [CoinGeckoAsyncApi](https://github.com/tosunthex/CoinGecko) to
provide up-to-date market information.

---

## Disclaimer

This plugin is developed by ModBros and is not officially affiliated with CoinGecko.  
Data is retrieved using the public CoinGecko API.
